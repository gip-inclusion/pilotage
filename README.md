# Le pilotage de l'inclusion

## Démarrer le projet

1. `source .venv/bin/activate`
2. `python manage.py runserver`

## Acces admin et DB

Pour l'accès à l'admin et pour récupérer la db (en sqlite3 pour l'instant), demander directement au patron
